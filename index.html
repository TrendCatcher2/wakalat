Paste your Google AI Studio API key to auto-fill judgments from a URL or text. The key stays in this browser only.
How to get a free Gemini 2.5 Flash key
  1. Open aistudio.google.com/app/apikey and sign in with Google.
  2. Click Create API key → choose or create a project.
  3. Copy the key that starts with AIza…
  4. Paste it above and press Save key. Done — the “Auto-fill with AI” button in Judgments now works.
